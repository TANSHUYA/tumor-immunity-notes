# 🧬 机体抗肿瘤免疫应答

---

## 一、细胞免疫

### 1. CD8⁺ 细胞毒性T淋巴细胞（CTL）

**识别：** pMHC-I  
**激活：** 由 DC + CD4⁺ Th 提供 **IL-2、IL-12**  
**杀伤机制：**
- **穿孔素 / 颗粒酶途径** → 裂解靶细胞  
- **Fas/FasL 途径** → 诱导凋亡  
- **TNF 途径** → 诱导凋亡  

---

### 2. CD4⁺ 辅助性T细胞（Th）

- **激活 CTL**
- **分泌细胞因子（CK）** → 激活巨噬细胞（Mφ）、NK细胞  
- **直接杀伤**（少数亚群具备）

---

### 3. NK 细胞（自然杀伤细胞）

**识别机制：** 识别 **缺乏 MHC I 分子的肿瘤细胞**  
**杀伤方式：**
- **自然杀伤**（无需抗原预先激活）  
- **ADCC（抗体依赖性细胞毒作用）**：通过 FcγRIII 介导  

---

### 4. 巨噬细胞（Mφ）

- **吞噬作用**  
- **分泌 TNF-α** → 抗肿瘤作用  
- **ADCC**  
- **抗原提呈功能（APC）**

---

### 5. γδ T 细胞

- **直接杀伤肿瘤细胞**  
- **分泌细胞因子**（如 IFN-γ）

---

## 二、体液免疫

### 1. 抗体的作用

| 功能 | 说明 |
|------|------|
| **中和作用** | 阻断肿瘤细胞表面毒性分子或配体 |
| **调理吞噬作用** | 通过 Fc 促进巨噬细胞吞噬 |
| **ADCC** | 结合 NK 细胞发挥杀伤作用 |
| **CDC（补体依赖性细胞毒作用）** | 激活补体系统溶解细胞 |
| **封闭受体** | 阻断生长因子受体（如 Her2） |

---

### 2. 补体系统的作用

- **CDC：** 通过膜攻击复合物（MAC）裂解肿瘤细胞  
- **调理作用（C3b）：** 增强吞噬细胞清除能力  

---

## 三、非特异性免疫

包括：
- **NK 细胞**
- **巨噬细胞**
- **γδ T 细胞**
- **细胞因子网络**

这些组成了**机体的天然防御系统**，可在无特异性抗原识别的情况下发挥抗肿瘤作用。

---

## 四、免疫调节与逃逸机制

### 1. 肿瘤免疫逃逸机制

| 类型 | 具体方式 |
|------|-----------|
| **抗原改变** | 抗原丢失或变异，逃避识别 |
| **MHC I 下调** | 减少CTL识别 |
| **缺乏共刺激信号** | 无法有效激活T细胞 |
| **分泌抑制因子** | TGF-β、IL-10 等抑制免疫反应 |
| **诱导AICD** | 促进活化T细胞凋亡 |
| **表达抗凋亡分子** | 如 Bcl-2 |
| **肿瘤微环境免疫抑制** | 存在 Treg、TAM、MDSC 等细胞群 |

---

### 2. 免疫治疗策略

| 策略 | 举例 |
|------|------|
| **免疫检查点抑制剂** | 抗 PD-1 / PD-L1、抗 CTLA-4 |
| **过继性细胞疗法** | CAR-T、TCR-T |
| **肿瘤疫苗** | DC 疫苗、肽疫苗等 |
| **细胞因子疗法** | IL-2、IFN-α 等 |
| **抗体靶向治疗** | Her2、CD20 等单抗 |
